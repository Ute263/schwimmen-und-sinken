# Mina und Toni forschen: Schwimmen und Sinken

Eine kleine statische Web-App als Quiz für Kinder am Ende der Klasse 1.

Die Kinder wiederholen spielerisch:

- Was schwimmt?
- Was sinkt?
- Warum sind Form, Luft und Auftrieb wichtig?
- Warum können große Schiffe schwimmen?

## GitHub Pages

Statische Web-App, geeignet für GitHub Pages.

Es gibt keine Anmeldung, keine Datenbank und keine externen Bibliotheken.

## Datenschutz

Keine Speicherung personenbezogener Daten.

Die App speichert keine Namen und keine Ergebnisse. Das Ergebnis wird nur direkt auf der Ergebnis-Seite angezeigt.

## Starten

Die Datei `index.html` im Browser öffnen.

Für GitHub Pages den ganzen Ordner mit `index.html`, `style.css`, `script.js`, `.nojekyll` und `assets/` hochladen.
